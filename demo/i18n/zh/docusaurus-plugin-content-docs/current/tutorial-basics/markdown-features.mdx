# Markdown功能

Docusaurus支持<strong><a href="https://daringfireball.net/projects/markdown/syntax">Markdown</a></strong>和一些<strong>额外的功能</strong>。

## 前面的内容

Markdown文档在顶部有称为<a href="https://jekyllrb.com/docs/front-matter/">Front Matter</a>的元数据。

```text title="my-doc.md"
// highlight-start
---
id: my-doc-id
title: My document title
description: My document description
slug: /my-custom-url
---
// highlight-end

## Markdown heading

Markdown text with [links](./hello.md)
```

## 链接

支持常规的Markdown链接，使用url路径或相对文件路径。

```md
Let's see how to [Create a page](/create-a-page).
```

```md
Let's see how to [Create a page](./create-a-page.md).
```

<strong>结果。</strong> Let's see how to <a href="./create-a-page.md">Create a page</a>.

## 图像

支持常规的Markdown图像。

你可以使用绝对路径来引用静态目录中的图像<code>（static/img/docusaurus.png</code>）。

```md
![Docusaurus logo](/img/docusaurus.png)
```

恐龙标志]\(/img/docusaurus.png)

<a href="../tutorial-extras/manage-docs-versions.md">你也可以参考相对于当前文件的图像，如</a>あるように、現在のファイルからの相対的な画像参照も可能です。

\##代码区块

支持Markdown代码块的语法高亮。

```jsx title="src/components/HelloDocusaurus.js"
function HelloDocusaurus() {
	return <h1>Hello, Docusaurus!</h1>;
}
```

```jsx title="src/components/HelloDocusaurus.js"
function HelloDocusaurus() {
	return <h1>Hello, Docusaurus!</h1>;
}
```

## 告诫

Docusaurus有一个特殊的语法来创建训诫和呼告。

::提示 我的提示

使用这个了不起的功能选项

:::

::危险 保重

这一行动是危险的

:::

::提示 我的提示

使用这个了不起的功能选项

:::

::危险 保重

这一行动是危险的

:::

## MDX和React组件

<a href="https://mdxjs.com/">MDX</a> can make your documentation more <strong>interactive</strong> and allows using any <strong>React components inside Markdown</strong>:

```jsx
export const Highlight = ({children, color}) => (
  <span
    style={{
      backgroundColor: color,
      borderRadius: '20px',
      color: '#fff',
      padding: '10px',
      cursor: 'pointer',
    }}
    onClick={() => {
      alert(`You clicked the color ${color} with label ${children}`)
    }}>
    {children}
  </span>
);

This is <Highlight color="#25c2a0">Docusaurus green</Highlight> !

This is <Highlight color="#1877F2">Facebook blue</Highlight> !
```
